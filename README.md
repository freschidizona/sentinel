## Getting Started

### Installation

1. Clone Repository
   ```sh
   git clone https://github.com/freschidizona/sentinel.git
   ```
2. Build Docker Services
   ```sh
   docker compose build
   ```
3. Run Docker Compose
   ```sh
   docker compose up
   ```
